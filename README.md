# Welcome to Mouse Island 🧀
 
A three-page website celebrating the world of Geronimo Stilton, built for SBA 307 (HTML & CSS).
 
## Description
 
Mouse Island is a small fan site that introduces visitors to Geronimo Stilton and his world. Explore the home page, meet the characters, and take a look at the map of Mouse Island. The site uses a cheese-themed background and a bright, playful color palette to match the spirit of the books.
 
## Pages
 
- **Home** (`index.html`) — welcome page with a featured image and navigation.
- **Meet the Characters** — profile cards for Geronimo, Creepella, and Thea, each with an image and short description.
- **Map of Mouse Island** — a look at the island and its key locations.
## Technologies Used
 
- **HTML5** — semantic structure, images, tables, and forms.
- **CSS3** — external stylesheet (`stylish.css`), plus inline and internal styles.
- **Flexbox** — used for the navigation bar and character card layout.
- **CSS animations** — `@keyframes` used to add motion to the pages.
## Features
 
- Consistent navigation across all three pages
- Responsive character cards built with Flexbox
- A cheese-textured background image
- A data table and interactive forms with a dropdown menu
- Animated elements for extra personality
## How to View
 
Download or clone the repository, then open `index.html` in any web browser.
